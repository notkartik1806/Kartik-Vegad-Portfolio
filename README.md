# Kartik Vegad - Portfolio

This is my professional portfolio built with **React** and **Vite**.

## Features
- **Modern UI/UX**: Built with Framer Motion for smooth animations and a premium feel.
- **Responsive Design**: Optimised for all device sizes.
- **Fast Performance**: Leveraging Vite for quick load times.

## Tech Stack
- React
- Vite
- Framer Motion
- CSS (Vanilla)

## Deployment
This project is automatically deployed to GitHub Pages via GitHub Actions.

## Local Development
1. Clone the repo
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
4. Build for production: `npm run build`
